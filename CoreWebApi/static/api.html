GET methods

  • /sales/ get list of sale items
  • /membersales/ get list of member-only sale items
  • /item/<upc> get info about item
  • /search/<term> get list of items matching the search term
  • /filehash/<filename> get MD5 hash of file
  • /file/<filename> get actual file

POST methods

  • /file/<apikey><file> Upload a file
